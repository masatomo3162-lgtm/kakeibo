# じぶん家計簿 Ver2.4.1

レシート・領収書を記録して、月別・年別・旧生活/新生活で支出を比較する個人用家計簿アプリです。
データはブラウザ内に保存されます。

## Ver2.4.1 の修正点

- 削除したカテゴリ・予算項目が再読み込み後に復活する問題を修正
- デフォルトカテゴリの改名で旧名が複製される問題を修正
- 分割レシートの1行編集でグループ情報が消える問題を修正
- 一括CSV復元で予算額・表示設定が戻らない問題を修正
- 貯金削除後に目標側へ連結情報が残る問題を修正
- CSVバックアップに並び順を追加

## Ver2.4.0 の主な機能

- 目標タブに「貯金先」を追加し、目的別貯金を複数選択可能に変更
- 目標進捗に、選択した複数の貯金先残高を反映
- 初期カテゴリを指定の20項目＋その他に刷新
- カテゴリの追加・編集・削除・上下移動に対応
- カテゴリと同名の予算項目がある場合、支出入力時に予算枠を自動選択
- カテゴリ名を変更した場合、同名の予算項目も同期
- 予算項目の初期値を8項目へ整理
- PC / iPad向けのアプリアイコンを更新
- Service Workerキャッシュをv2.4.0に更新

## 注意

同じ貯金先を複数の目標に紐づけると、各目標の進捗に同じ残高が反映されます。
実際のお金が増えるわけではないため、基本は「1つの貯金先＝1つの主目標」で使うのがおすすめです。

古いバージョンが表示される場合は、ブラウザの Service Worker / Cache Storage を削除して再読み込みしてください。
