# じぶん家計簿 v1.0.0

個人用の広告なし家計簿アプリです。データはブラウザのローカルストレージに保存されます。

## v1.0.0 の主な変更

- 正式版としてバージョンを v1.0.0 に更新
- 実績解除の解除数表示を大きくして視認性を改善
- 実績名と条件をよりユニークな内容に刷新
- 実績カードのアイコン表示を強化し、解除済みにはチェックマークを表示
- アプリアイコンを刷新
- 設定画面に更新履歴を追加
- 既存の v0.5.0 データがあれば自動移行

## 主な機能

- 支出入力、履歴、月別集計
- 支払方法：現金、キャッシュレス、引き落とし
- 支払元：現金、現金封筒、銀行口座、クレジットカード、電子マネー・QR
- クレジットカード初期候補：リクルートカード、楽天カード、KABUカード、セブンカード、PayPayカード、三井住友カード、LIFEカード、ドンペンカード（UCS）、JCB系カード（要確認）
- クレジットカードの引き落とし先は楽天銀行
- 目標管理
- 楽天銀行CSVからカード引き落としを精算記録として登録
- 実績解除と今月の称号
- 支出CSV、支払元CSV、目標CSV、カード精算CSVの書き出し
- 支出CSV、目標CSV、楽天銀行CSVの読み込み
- 設定画面でカテゴリ管理と更新履歴確認

## 使い方

1. `index.html` をブラウザで開く
2. 入力タブで支出を保存
3. 支払元タブで現金封筒・銀行・カードを編集
4. 銀行CSVタブで楽天銀行CSVを読み込み、カード引き落としを精算記録として保存
5. 実績タブで解除状況と次に狙う実績を確認
6. 集計タブで月別の支出・カード未払・カード精算・資産反映プレビューを確認
7. 必要に応じてCSVを書き出し

## 注意

- KABUカードの楽天銀行側の明細文言は未確認です。初回引き落とし後、`入出金内容` の文言を追加してください。
- `（カ）シ゛エ－シ－ヒ゛－` はリクルートカード・セブンカードのどちらかをCSVだけでは判別できません。いったんJCB系カード（要確認）に集約します。
- カード引き落としは通常の支出には入れません。カード利用時点で支出計上し、引き落とし時点では未払の精算として扱います。
- 実績は支出額の多さではなく、記録の継続・分類・確認を評価する設計です。
- 総資産管理アプリ本体との完全統合はまだ未実装です。
