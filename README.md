# じぶん家計簿 Ver2.1.0

広告なし・端末保存の個人用家計簿アプリです。

## 方針

総資産管理との自動連携はせず、レシート・領収書を記録して「今月いくら使ったか」「先月・去年・旧生活／新生活でどう変わったか」を把握することに絞っています。

## 主な機能

- 支出入力：日付、金額、カテゴリ、お店・支払先、支払方法、生活フェーズ、メモ
- 新規カテゴリ・新規お店の入力画面からの追加
- カテゴリ／お店の設定画面での追加・編集・削除
- 月／年／生活フェーズの左右比較
- カテゴリ別・お店別・支払方法別比較
- 履歴の編集・削除・フィルター
- 目標管理
- 目的別封筒貯金／目的別銀行貯金
- 実績解除
- 一括CSVバックアップ／復元

## 注意

データはブラウザ内に保存されます。端末変更やキャッシュ削除の前に、一括CSVを書き出してください。
