# じぶん家計簿 v2.0.0

レシート・領収書の支出記録に絞った、個人用の家計簿アプリです。

## v2.0.0 の方針

- 総資産管理アプリとの自動連携は中止
- クレジットカード未払・銀行CSV精算・電子マネー残高管理は廃止
- 実際に買った・支払ったタイミングの支出だけを記録
- 支払方法は「現金 / カード / 電子マネー / 口座引き落とし / その他」だけ
- 月別、年別、生活フェーズ別の左右比較を重視

## 主な機能

- 支出入力
- 履歴の編集・削除
- カテゴリ追加・編集・削除
- 左右比較
  - 今月 vs 先月
  - 今年 vs 去年
  - 単身期 vs 家族生活期
- 生活フェーズと生活人数の記録
- 目標管理
- 目的別封筒貯金 / 目的別銀行貯金
- 目標と貯金の連結
- 実績解除
- 支出 / 目標 / 貯金 CSV入出力
- ブラウザ内ローカル保存
- PWA対応

## 入力ルール

チャージやカード引き落としは原則として支出にしません。
実際に買った・支払ったときだけ入力します。

例：
- クレカで買い物 → 入力する
- クレカ引き落とし → 入力しない
- nanacoチャージ → 入力しない
- nanacoで公共料金支払い → 入力する
- スーペイチャージ → 入力しない
- スーペイで食品購入 → 入力する

## バックアップ

データはブラウザのローカルストレージに保存されます。
端末変更やブラウザ初期化に備えて、定期的にCSVを書き出してください。
