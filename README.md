# じぶん家計簿 Ver2.2.0

レシート・領収書を記録して、月別・年別・旧生活/新生活で支出を比較する個人用家計簿アプリです。

## Ver2.2.0 の主な機能

- 予算管理タブを追加
- お小遣い・食費・娯楽費などの月予算を管理
- 使用率70% / 90% / 100%超過で段階警告
- 支出入力に「お小遣いから支出」と「予算枠」を追加
- レシート分割入力を追加
- レシート合計・内訳合計・差額の照合表示
- 予算項目を設定画面で追加・編集・削除
- 一括CSVバックアップに予算データを追加
- 予算関連・分割入力関連の実績解除を追加
- アプリアイコンを新デザインに更新

## 使い方

`index.html` をブラウザで開くだけで使えます。データはブラウザ内に保存されます。

GitHub Pages にアップロードする場合は、フォルダ内の全ファイルを同じ階層に置いてください。

## 注意

古いバージョンが表示される場合は、ブラウザの Service Worker / Cache Storage を削除して再読み込みしてください。
