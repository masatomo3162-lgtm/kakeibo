# Changelog

## v0.5.0 - 2026-07-07

- 「実績」タブを追加
- 7月開始、初入力、入力件数、記録日数、連続記録、カテゴリ数、支払元数、メモ件数を自動判定
- カード支出、楽天銀行CSVのカード精算、目標登録、目標達成、ウイスキー・ゲームカテゴリの実績を追加
- 解除数、対象月の記録日数、対象月の入力件数、最長連続記録を表示
- 今月の称号と次に狙える実績を表示
- v0.4.1データ移行処理を追加

## v0.4.1 - 2026-07-07

- 楽天銀行CSVの実明細に合わせてクレカ引き落とし判定を更新
- `ミツイスミトモカ－ト゛　（カ` → 三井住友カード
- `ラクテンカ－ト゛サ－ヒ゛ス` → 楽天カード
- `ＰＡＹＰＡＹカ－ト゛` → PayPayカード
- `ライフカ－ト゛（カ` → LIFEカード
- `カ　ユ－シ－エス` → ドンペンカード（UCS）
- `（カ）シ゛エ－シ－ヒ゛－` → JCB系カード（要確認）として登録
- 楽天銀行CSV特有の半角・全角・長音・分離濁点を吸収する文字正規化を強化
- 三井住友カード、LIFEカード、ドンペンカード（UCS）、JCB系カード（要確認）を初期支払元に追加
- v0.4.0データ移行処理を追加

## v0.4.0 - 2026-07-07

- 「銀行CSV」タブを追加
- 楽天銀行CSVからカード引き落とし行を検出する機能を追加
- クレジットカード引き落としを支出ではなくカード未払の精算として保存する設計に変更
- 楽天カード、リクルートカード、KABUカード、セブンカード、PayPayカードの判定キーワードを追加
- カード精算記録の一覧、削除、CSV書き出しに対応
- 集計タブにカード精算額を追加
- v0.3.0データ移行処理を追加

## v0.3.0 - 2026-07-07

- 登録済み支払元のカード表示が縦書き・文字詰まりになる問題を修正
- 支払元名を横書きで読めるように表示レイアウトを調整
- 「目標」タブを追加
- 目標金額、現在の積立額、目標時期、保管先、優先度、メモを登録できる機能を追加
- 目標カードで進捗率、残額、月々必要額を表示
- 目標CSVの書き出し・読み込みを追加
- v0.2.0データ移行処理を追加

## v0.2.0 - 2026-07-07

- 支払元を総資産管理アプリ連携前提の「資産口座マスター」に変更
- 種別に現金・現金封筒・銀行口座・クレジットカード・電子マネーを追加
- リクルートカード、楽天カード、KABUカード、セブンカード、PayPayカードを初期登録
- クレジットカードの引き落とし先を楽天銀行に設定
- 月別集計にカード未払、即時減少、資産反映プレビューを追加
- 支払元CSV出力を追加
- 支出CSVの列を総資産連携向けに拡張
- v0.1.0データ移行処理を追加

## v0.1.0 - 2026-07-07
