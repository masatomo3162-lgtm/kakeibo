# 更新履歴

- 総資産連携方針を停止

- 支出記録を「レシート・領収書」中心に再設計
- 支払元マスターを廃止し、支払方法だけに簡素化
- 左右比較タブを新設
- 月別、年別、生活フェーズ別比較を実装
- 生活人数の記録に対応
- 目的別封筒貯金、目的別銀行貯金を追加
- 貯金と目標の連結機能を追加
- 実績内容を支出比較版に刷新
- アイコンを支出比較アプリ向けに再作成
- 設定画面に更新履歴を継続掲載
- v1.0.0以前の支出・目標データを可能な範囲で移行
